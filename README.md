# Option Pricing Tools

This repository includes:

- A Cox-Ross-Rubinstein binomial tree pricer for European and American calls/puts, including delta estimation.
- A Monte Carlo estimator for options under geometric Brownian motion where you can supply any custom payoff function.

## Quick start

Run the modules directly to see example binomial and Monte Carlo price calculations:

```bash
python binomial_option_pricing.py
python monte_carlo_option_pricing.py
```
